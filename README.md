# Teste Técnico – Landing Page

Utilizei **ChatGPT** e **Claude** como apoio para estruturar o layout inicial, principalmente interpretando o design do Figma e gerando uma base de HTML/CSS.

A IA ajudou principalmente na **conversão inicial do layout (Figma → código)** e na organização de estilos, acelerando o início do desenvolvimento.

Realizei diversos ajustes manuais para garantir fidelidade ao design, como **espaçamentos, tipografia e responsividade**, já que a saída inicial não ficava 100% precisa.

Na Parte 1, implementei também um **menu mobile responsivo**, que não estava completamente definido, para melhorar a experiência em telas menores.

Usei o PNG e as propriedades de CSS do Figma como referência para refinar o resultado final.
